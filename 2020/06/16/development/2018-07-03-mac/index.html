Jun 16 2020 Development

mysql workbench 단축키

  1. 현재 커서가 위치하는 쿼리 실행 : Ctrl + Enter
  2. 쿼리 보기좋게 정렬하기(Beautify Query) : Ctrl + B
  3. 세미클론(;)으로 분리된 쿼리 다중(여러개) 실행 : Ctrl + Shift + Enter
  4. 새로운 쿼리 에디터 열기 : Ctrl + T (Ctrl키 + t)
  5. 저장된 스크립트파일(*.sql)파일 열기 : Ctrl + Shift + O (Ctrl키 + Shift키 + o)
  6. 쿼리를 보기 좋게 정렬 : Ctrl + B (Ctrl키 + b)
  7. Reverse Engineer Database : Ctrl + R (Ctrl키 + r)

mac OS 단축키

  1. 숨긴파일 확인 shift + command + .
  2. 켜져있는 프로그램 한눈에 보기 control + 방향키(위로)
  3. 프로그램 종요 : command + w
  4. 프로그램 숨기기 : command + h
