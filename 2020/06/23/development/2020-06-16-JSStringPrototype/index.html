JS String.prototype

Jun 23 2020 Development

JS String.prototype 중 유용한 몇가지 정리

  1. String.chartAt() 인덱스의 character 리턴
    만약 인덱스가 문자열 길이보다 큰 경우 빈 문자열 “ “ 을 리턴

  1. String.split()
    str.split([separator[, limit]]) 문자열을 구분자를 이용하여 배열로 리턴 인덱스를 넣어 limit

  1. String splice()
    str.slice(beginIndex[, endIndex])
    index를 지정하여 문자열을 리턴

참조 페이지 https://developer.mozilla.org/ko/docs/Web/JavaScript/Reference/
